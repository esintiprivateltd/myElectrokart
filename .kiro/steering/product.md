# Product Overview

This is an e-commerce web application built with modern React technologies. The application features:

- Product browsing and catalog with categories
- Product detail pages with full specifications
- Shopping cart functionality with persistent state
- Checkout flow and order completion
- Informational pages (About, Help, Contact, Privacy)
- Responsive design with mobile support
- WhatsApp integration for customer support

The project was initially scaffolded using Lovable.dev and is designed for easy deployment and iteration.
